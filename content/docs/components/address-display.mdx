## Usage

### Basic

```tsx
<AddressDisplay address="7xKpR4nm3kW9vBzL5hQd2mFnZq8gT4pYx9eRwVb3mKs" />
```

### With Explorer Link

```tsx
<AddressDisplay
  address="7xKpR4nm3kW9vBzL5hQd2mFnZq8gT4pYx9eRwVb3mKs"
  explorerUrl="https://solscan.io/account"
/>
```

### Custom Truncation

```tsx
<AddressDisplay
  address="7xKpR4nm3kW9vBzL5hQd2mFnZq8gT4pYx9eRwVb3mKs"
  truncateChars={[8, 8]}
/>
```
