## Usage

```tsx
<OrderFormCard
  title="Edit TP/SL"
  description="Adjust the parameters on your order"
  entryPrice={148.32}
  details={[
    { label: "Size", value: "150.0" },
    { label: "Entry Price", value: "$148.32" },
    { label: "Mark Price", value: "$162.56" },
    { label: "Liquidation Price", value: "$118.66" },
  ]}
  onSubmit={(values) => {
    console.log("TP:", values.tpPrice, "TP%:", values.tpPercent);
    console.log("SL:", values.slPrice, "SL%:", values.slPercent);
  }}
/>
```
