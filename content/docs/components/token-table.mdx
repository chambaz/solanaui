## Usage

```tsx
<TokenTable
  tokens={[
    { symbol: "SOL", icon: "/sol.png", price: "$162.56" },
    { symbol: "USDC", icon: "/usdc.png", price: "$1.00" },
  ]}
/>
```

### With Actions

Pass an `actions` array of React nodes to render action buttons for each row. The array should match the length of the `tokens` array.

```tsx
import { Button } from "@/components/ui/button";

<TokenTable
  tokens={[
    { symbol: "SOL", icon: "/sol.png", price: "$162.56", apy: "6.82%" },
    { symbol: "USDC", icon: "/usdc.png", price: "$1.00", apy: "8.45%" },
  ]}
  actions={[
    <Button variant="outline" size="sm">Deposit</Button>,
    <Button variant="outline" size="sm">Deposit</Button>,
  ]}
/>
```
