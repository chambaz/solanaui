### Custom Series Options

Customize individual series appearance:

```tsx
<TradeChart
  series={[
    {
      data: bitcoinData,
      options: {
        lineWidth: 3,
        priceLineVisible: false,
      },
    },
    {
      data: ethereumData,
      options: {
        lineWidth: 2,
      },
    },
  ]}
  chartOptions={{
    grid: {
      vertLines: { visible: false },
      horzLines: { visible: false },
    },
  }}
/>
```

## Props

### TradeChart

| Prop           | Type                            | Default              | Description                            |
| -------------- | ------------------------------- | -------------------- | -------------------------------------- |
| `series`       | `SeriesConfig[]`                | Required             | Array of series configurations         |
| `className`    | `string`                        | `"h-[320px] w-full"` | Tailwind classes for container styling |
| `chartOptions` | `DeepPartial<TimeChartOptions>` | `{}`                 | Custom chart configuration             |

### SeriesConfig

| Prop      | Type                                    | Description                       |
| --------- | --------------------------------------- | --------------------------------- |
| `data`    | `SingleValueData[]`                     | Array of time/value data points   |
| `options` | `DeepPartial<AreaSeriesPartialOptions>` | Custom styling options for series |
