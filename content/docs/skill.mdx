The SolanaUI skill gives AI coding agents (Claude Code, Cursor, Windsurf, etc.) complete knowledge of the component library -- all 30 components with full TypeScript prop interfaces, usage examples, composition patterns for common Solana app layouts, and token image CDN URLs.

Once installed, your agent can generate production-quality Solana app UIs using the SolanaUI component library without any additional context or prompting.

## Install via npx (Recommended)

<Tabs items={["npx", "Manual"]}>
  <Tab value="npx">
    ```bash
    npx skills add chambaz/solanaui
    ```

    This installs the skill into your project's `.claude/skills/` directory where Claude Code and other compatible agents will automatically pick it up.
  </Tab>
  <Tab value="Manual">
    Create a file at `.claude/skills/solanaui.md` in your project root and copy the full skill content from the [published skill on skills.sh](https://skills.sh/chambaz/solanaui).

    The skill file should be committed to your repository so all team members and CI environments have access to it.
  </Tab>
</Tabs>

## What's Included

The skill provides your agent with:

- **30 component definitions** with full TypeScript prop interfaces and usage examples
- **Composition patterns** for building common Solana app layouts (swap interfaces, perps trading pages, lending dashboards, liquidity pools, NFT marketplaces, token portfolios, DeFi stats dashboards, staking pages)
- **shadcn/ui primitives** list so the agent knows which base components are available
- **Output rules** ensuring generated code follows SolanaUI conventions (Tailwind CSS, dark theme compatible, no SDK dependencies, static demo data)
